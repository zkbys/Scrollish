# Smart Dictionary Feature - 使用指南

## 功能概述

智能词典功能为 Scrollish 应用添加了强大的单词学习能力：

1. **📚 词书管理**：支持系统内置词书（如四级、六级、雅思等）和用户自定义词书
2. **✨ 智能高亮**：自动识别并高亮文章中出现在当前词书中的单词
3. **🔍 点击查词**：点击任何单词即可查看释义（词书中的单词有详细解释和例句）
4. **🔄 动态切换**：可随时切换不同的词书，高亮内容会实时更新

## 快速开始

### 1. 数据库初始化

确保你已经在 Supabase 后台执行了 `supabase_dictionary_schema.sql` 中的 SQL 语句。

### 2. 填充测试数据

为了快速测试功能，我们提供了一个开发工具：

1. 在应用中，**点击屏幕左上角（隐形区域）**
2. 会弹出开发工具面板
3. 点击 **"Seed Dictionary Data"** 按钮
4. 等待提示"Dictionary seeded!"

这会在数据库中创建一个测试词书，包含以下单词：
- community, discussion, reply, message, thread
- context, explore, profile, trending, awesome

### 3. 体验功能

1. 进入 **ChatRoom**（讨论区）
2. 你会看到消息中的某些单词被**橙色下划线**标注
3. **点击高亮的单词**，会弹出底部卡片显示：
   - 音标
   - 中文释义
   - 例句（英文 + 中文）
4. 点击**普通单词**，会显示"即将支持通用查词"的占位界面

## 技术架构

### 核心组件

1. **`SmartText.tsx`**
   - 文本分词组件
   - 自动识别单词并与词库比对
   - 渲染高亮效果

2. **`WordDetailSheet.tsx`**
   - 底部弹窗组件
   - 显示单词详细信息
   - 支持 VIP 词（词书中）和普通词的区分

3. **`useDictionaryStore.ts`**
   - Zustand 状态管理
   - 管理词书列表、当前激活词书
   - 维护高亮词集合（Set 结构，O(1) 查询）

### 数据库表

- **`dictionaries`**：词书元数据（名称、类型、是否公开）
- **`dictionary_entries`**：词条详情（单词、释义、音标、例句）

### 添加更多词书

你可以通过以下方式添加新词书：

1. 准备 JSON 或 CSV 格式的单词表
2. 使用类似 `seedDictionary.ts` 的脚本导入
3. 或者在应用中提供上传功能（未来版本）

## 性能优化

- **词库加载**：只加载单词列表（不含释义），减少网络传输
- **内存缓存**：使用 `Set` 数据结构，查询速度极快
- **按需加载**：只在点击时才获取完整的单词详情
