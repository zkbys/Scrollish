# 词库导入指南 - KyleBing 开源词库

## 📚 词库来源

本项目使用了 [KyleBing/english-vocabulary](https://github.com/KyleBing/english-vocabulary) 开源词库，包含：

- **CET-4** (大学英语四级) - 约 2500 词
- **CET-6** (大学英语六级) - 约 2000 词  
- **考研** (Postgraduate) - 约 5500 词
- **TOEFL** (托福) - 核心词汇
- **IELTS** (雅思) - 核心词汇
- **SAT** - 核心词汇

## 🚀 快速导入

### 方法一：使用 Dev Tools（推荐）

1. **打开开发工具**
   - 点击屏幕**左上角**（隐形区域）
   - 会弹出 Dev Tools 面板

2. **选择导入方案**
   
   根据你的需求选择：

   - **📚 College (CET-4/6)** - 大学生常用
     - 包含：四级 + 六级
     - 约 4500 词
     - 导入时间：1-2 分钟

   - **🎓 Postgraduate** - 考研学生
     - 包含：四级 + 六级 + 考研
     - 约 10000 词
     - 导入时间：2-3 分钟

   - **✈️ Study Abroad** - 出国留学
     - 包含：TOEFL + IELTS + SAT
     - 导入时间：2-3 分钟

   - **🚀 Import All** - 全部导入
     - 包含：所有词书
     - 约 15000+ 词
     - 导入时间：3-5 分钟

3. **等待导入完成**
   - 点击按钮后会有确认提示
   - 导入过程中请**不要关闭页面**
   - 完成后会显示导入的单词数量

4. **切换词书**
   - 进入 **Study** 页面
   - 点击橙色卡片中的 **"Switch"** 按钮
   - 选择你想要使用的词书

### 方法二：使用控制台（高级）

如果你熟悉浏览器控制台，可以直接调用：

```javascript
// 在浏览器控制台中执行

// 导入大学生常用词书
await importPresets.college()

// 导入考研词书
await importPresets.postgraduate()

// 导入出国留学词书
await importPresets.abroad()

// 导入全部
await importPresets.all()
```

## 📊 导入后的效果

导入成功后，你会看到：

1. **Study 页面**
   - 词书选择器中会出现新的词书
   - 可以自由切换不同的词书

2. **Home 和 ChatRoom 页面**
   - 文章中的单词会被自动高亮（橙色下划线）
   - 点击高亮单词可以查看详细释义

3. **单词详情**
   - 词性和中文释义
   - 常用短语和搭配
   - 点击发音按钮可以听读音

## 🔧 技术细节

### 数据格式

导入的每个单词包含：

```typescript
{
  word: "ability",              // 单词
  definition: "n. 能力，能耐；才能",  // 释义
  detail: {
    phonetic: "",               // 音标（此数据集不包含）
    examples: [                 // 例句（从短语转换）
      {
        en: "innovation ability",
        zh: "创新能力"
      }
    ]
  }
}
```

### 导入流程

1. **创建词书记录**：在 `dictionaries` 表中创建词书元数据
2. **获取词汇数据**：从 GitHub 获取 JSON 格式的词汇数据
3. **批量插入**：每次插入 500 个单词（Supabase 限制）
4. **进度反馈**：在控制台显示导入进度

### 性能优化

- **批量插入**：使用 500 词/批次的策略，平衡速度和稳定性
- **错误处理**：单个批次失败不影响整体导入
- **进度显示**：实时显示已导入的单词数量

## ⚠️ 注意事项

1. **网络要求**
   - 需要能访问 GitHub raw 内容
   - 如果网络不稳定，可能导致导入失败

2. **时间消耗**
   - 导入过程需要几分钟时间
   - 请耐心等待，不要中途关闭页面

3. **重复导入**
   - 目前不会检查重复
   - 重复导入会创建新的词书记录

4. **数据库容量**
   - Supabase 免费版有存储限制
   - 建议只导入需要的词书

## 🐛 常见问题

### Q: 导入失败怎么办？

A: 可能的原因：
- 网络问题：检查是否能访问 GitHub
- 数据库问题：检查 Supabase 连接
- 查看浏览器控制台的错误信息

### Q: 导入后看不到高亮？

A: 检查：
1. 是否在 Study 页面选择了词书
2. 刷新页面让词库生效
3. 查看控制台是否有错误

### Q: 可以删除词书吗？

A: 目前需要手动在 Supabase 后台删除：
1. 打开 Supabase Dashboard
2. 进入 `dictionaries` 表
3. 删除对应的记录（会级联删除所有单词）

### Q: 可以自定义词书吗？

A: 可以！参考 `seedDictionary.ts` 的格式，创建自己的词书数据。
