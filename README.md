# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1AdphHLASOtdSd1HszgSwtLuQPPWvWnvc

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `VITE_SILICONFLOW_API_KEY` in [.env](.env) to your Silicon API Key. Silmiar to `.env_example`.
3. Run the app:
   `npm run dev`
