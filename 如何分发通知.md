# 如何分发通知 (Notification Delivery Guide)

由于系统是基于 `notifications` 表驱动的，你可以通过以下三种方式发送通知：

## 1. 命令行/SQL 编辑器 (手动分发)
适用于测试或手动向所有用户发布公告。请在 Supabase SQL Editor 中运行：

### A. 向特定用户发送
```sql
insert into public.notifications (user_id, type, title, content)
values ('用户_UUID_在这里', 'system', '欢迎回来', '很高兴再次见到你！');
```

### B. 向所有用户广播 (Broadcast)
```sql
insert into public.notifications (user_id, type, title, content)
select id, 'system', '维护通知', '系统将于今晚进行例行维护。'
from public.profiles;
```

---

## 2. 前端代码实现 (应用内触发)
如果你想在用户完成某个操作（如获得金币）时触发通知，可以使用 `supabase` 客户端：

```typescript
import { supabase } from '../supabase'

/**
 * 发送通知的工具函数
 */
export const sendNotification = async (userId: string, title: string, content: string, type: 'system' | 'award' | 'social' = 'system') => {
  const { error } = await supabase
    .from('notifications')
    .insert([{
      user_id: userId,
      title,
      content,
      type
    }])
    
  if (error) console.error('发送通知失败:', error.message)
}

// 示例：用户获得奖励时调用
// sendNotification(profile.id, '金币到账', '你刚刚获得了 10 哆吧币！', 'award')
```
