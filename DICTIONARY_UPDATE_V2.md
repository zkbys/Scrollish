## 🚀 使用指南

### 切换词书
1. 进入 **Study** 页面（底部导航第三个图标）
2. 点击顶部橙色卡片中的 **"Switch"** 按钮
3. 在弹出的列表中选择你想要的词书
4. 返回 Home 或 ChatRoom，高亮的单词会立即更新

### 查词体验
1. 在 **Home** 页面浏览帖子时，注意标题中的橙色下划线
2. 点击任何高亮的单词，会弹出详细释义
3. 在释义卡片中：
   - 点击 **"Save Word"** 保存到复习列表
   - 点击 **🔊 图标** 听单词发音

### 学习统计
- **Words Learned**: 显示你已经学习的单词总数
- **Daily Streak**: 显示连续学习天数
- **Weekly Progress**: 每周学习进度条形图

## 🔧 技术亮点

### 1. 性能优化
- **词库缓存**：词书的单词列表加载后存储在内存中（Set 结构）
- **O(1) 查询**：使用 Set 进行单词匹配，速度极快
- **按需加载**：只在点击时才获取完整的单词详情

### 2. 用户体验
- **热更新**：切换词书后，所有页面的高亮立即更新
- **无缝集成**：SmartText 组件可以轻松集成到任何文本显示位置
- **优雅降级**：如果词书为空或加载失败，不影响正常使用

### 3. 可扩展性
- **模块化设计**：SmartText 和 WordDetailSheet 是独立组件
- **易于集成**：只需两行代码即可在新页面添加智能查词
- **数据驱动**：词书数据完全由数据库管理，易于扩展
